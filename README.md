🤖 AI Driven Code Reviewer

An AI-powered web application that analyzes multiple programming language code and provides instant feedback on errors, code quality and optimizations.

📌 What it does

1. Detects syntax errors using Python AST parsing
2. Identifies bugs like unused imports, undefined variables, unreachable code
3. Performs PEP8 style analysis and assigns a quality score
4. Generates AI-based suggestions and optimized code using Groq
5. Shows time complexity insights and improved code snippets

⚙️ How it works

1. User pastes code or uploads a file
2. System runs AST parsing + rule-based checks
3. AI model (Groq) analyzes code for improvements
4. User receives: Errors & issues, Code quality score, Time complexity insights, Optimized code suggestions

🛠️ Tech Stack

* Python
* Reflex (UI + Backend)
* AST (Code parsing)
* Groq API (AI suggestions)
* PEP8 (Style analysis)
